Phase 2 拆分为 Phase 2a（优先）和 Phase 2b（其余）。

Phase 2a 四大功能：
1. AI角色扩展 — 初级段(冬冬/狸花花/铁墩墩) + 中级段(银鬃/咕噜/云朵师父)
2. Level 2-3 课程制作（各约15课）
3. 弱点诊断系统（分析对弈+做题数据，推荐专项训练）
4. 自适应难度算法优化（动态调整引擎参数，目标胜率40%-60%）

**Why:** 这4项是用户从入门段升级到中级段的核心路径依赖，缺少任何一项都会导致用户成长链断裂。
**How to apply:** 所有Phase 2a的开发优先于Phase 2b功能。规划文档在 docs/phase2a-plan.md。
