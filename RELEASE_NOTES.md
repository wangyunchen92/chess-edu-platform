## v0.2.0 — 学习模块打磨 + 数据联通 (2026-03-29)

**状态**: 已完成

### 学习模块卡通课堂改造
- 全新左右分栏布局（左棋盘 + 右聊天），沉浸式全屏教学
- 双角色对话系统：豆丁老师 + 小琪同学，微信风格聊天流
- Level 0 全部10课重写为故事线驱动对话（260+ steps）
- 13个新课堂组件（TeacherAvatar、ChatMessage、ChatPanel、StarProgress等）
- 6套课程主题配色，每课视觉不同
- 棋子走法课增加实际走子练习（19个，含吃子/升变/易位）
- 兵升变选择弹窗（后/车/象/马）
- 走子/吃子/将军/易位音效（Lichess标准音频）
- 语音播放系统（Web Speech API + ChatTTS 预留接口）

### 数据联通修复
- GamePage 对局结束调用 `completeGame` API，记录到后端
- Dashboard 返回时自动刷新（window.focus 监听）
- 本周数据概览从真实数据计算（不再用 mock）
- 对弈/谜题完成后触发成就检查
- 每日谜题 attempted 状态正确读取

- rank_title 中文翻译（apprentice_1 → 学徒 I）
- 经验值进度条修正（等级内进度，非 xp_today）

### Bug 修复
- Vite 代理端口 8001→8000
- quiz 答案映射缺 `correct` 字段
- 点击格子互动无响应（Chessboard 加 onSquareClick）
- 多格点选支持（如"点击兵能走到的所有格子"）
- Puzzle Rating 改中文
- 谜题主题 mate_in_1 翻译为中文题目要求
