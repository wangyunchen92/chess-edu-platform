## v0.3.0 — 后台管理 + 数据质量 + 训练联动 (2026-03-30)

**状态**: 已发布

### 后台管理系统
- 管理员控制台：数据概览 / 账号管理 / 会员管理 / 积分经验管理
- 7个新API + 8个Pydantic Schema
- 非admin用户403拦截，禁用用户JWT立即失效
- 积分调整写入审计日志（rating_histories）

### 谜题库替换
- 全部160道谜题替换为 Lichess 开源数据库精选（600万中筛选）
- chess.js 100%验证通过（FEN合法+走法正确+mate题确认将杀）
- 覆盖主题：一步杀/两步杀/底线杀/捉双/牵制/串击/弃子/引离等

### 段位系统统一
- 去掉XP等级系统（Lv.X + 经验条），改用纯段位制
- 7级段位：学徒→棋手→战士→骑士→大师→宗师→传奇（各3小段）
- 对弈评分和谜题评分独立显示

### 训练自动完成
- 做完3道每日谜题 → 自动标记"战术热身"完成
- 完成一节课程(100%) → 自动标记"继续学习"完成
- 完成一局对弈 → 自动标记"实战对弈"完成
- 修复 SQLAlchemy JSON mutation 不持久化（flag_modified）

### 数据质量
- 清除全部9个页面的假MOCK数据（1200分/42局/73%正确率等）
- API失败显示错误提示+重试按钮，不再静默显示假数据
- 首页/个人中心评分统一取同一数据源
- 连续登录streak在Dashboard接口自动更新（不依赖重新登录）

### Bug修复
- 谜题升变走法匹配（handleMove加promotion参数）
- 限流提高至600rpm（开发环境）
- DailyPlanPage无限循环请求（useCallback依赖修复）
- 每日谜题attempted状态正确读取
- Admin角色自动premium权限（不受配额限制）
- 棋盘/音效/Stockfish路径加BASE_URL前缀
- 导航栏"我的"恢复+退出登录，"设置"仅admin可见
- 对局复盘从PGN重建走法（不依赖review_data）
